prashpyutil: Utils for Prashant
==========================

**prashpyutil** is an elegant and simple util library for Python, built for Prashant.

-------------------

**Behold, the power of Utils**::

    >>> r = prashpyutil.sumoftwo(3, 2)
    >>> r
    5
